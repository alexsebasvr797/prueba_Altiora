# QA Automation - Playwright

## 📌 Descripción
Este proyecto contiene pruebas automatizadas E2E para validar la funcionalidad de ordenamiento de productos por precio en un ecommerce demo.

## 🎯 Objetivo
Validar que los productos se ordenan correctamente de menor a mayor precio cuando el usuario selecciona la opción correspondiente.

---

## 🧠 Justificación de Automatización

Se decidió automatizar esta funcionalidad porque:

- Es un flujo crítico del negocio que impacta la experiencia del usuario.
- Es susceptible a regresiones cuando se modifican listas o filtros.
- Tiene criterios claros y verificables.
- Permite reducir el tiempo de validación manual.

---
